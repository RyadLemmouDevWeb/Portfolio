## 🚀 Installation & Développement

### **Prérequis**
- Node.js (version 16+)
- npm ou yarn

### **Installation**
```bash
git clone git@github.com:RyadLemmouDevWeb/Portfolio.git

cd Portfolio/@

npm install

npm run dev
```

### **Scripts disponibles**
```bash
npm run dev      
npm run build    
npm run preview  
npm run lint     
```

## 🎨 Fonctionnalités Techniques

### **Animations Avancées**
- **Scroll-triggered animations** avec Framer Motion
